🛡️ Fault Tolerance Mechanisms
Active
💓
Heartbeat Monitoring
Nodes report every 2s. Declared DEAD after 6s silence. Automatic re-replication triggered.
📋
Multi-Node Replication
Each file chunk is replicated to N nodes (configurable). Maintains availability during failures.
🔐
SHA-256 Integrity
Every chunk checksum is verified on read. Corrupted data is detected and flagged immediately.
⚡
Automatic Failover
Downloads automatically retry across all healthy replicas. Zero manual intervention required.
🔄
Auto Re-replication
Under-replicated chunks are automatically redistributed to healthy nodes when a failure is detected.
🌐
Dynamic Scaling
Add/remove nodes at runtime. New nodes join the cluster immediately and accept new replication targets.
